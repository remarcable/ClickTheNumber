# [Click The Number!](http://marcnitzsche.de/ClickTheNumber/)

Test your reaction time with this fun game.

+-+

Click on the boxes with the numbers from 1 to 25 in ascending order.

```

[ 21 ] [ 17 ] [ 22 ] [ 14 ] [ 12 ]
[  9 ] [ 23 ] [ 11 ] [ 25 ] [ 18 ]
[ 13 ] [ 15 ] [ 19 ] [  7 ] [ 24 ]
[  8 ] [ 20 ] [  1 ] [  5 ] [  6 ]
[  4 ] [  3 ] [ 10 ] [  2 ] [ 16 ]

```
